Legal Notice Generator

আইনি নোটিশ জেনারেটর

If a customer abandons their outstanding debt and becomes uncontactable, the system can generate a formal "Notice of Default" PDF — a legally structured document citing the customer's name, total outstanding amount, transaction dates, and shop details — ready for submission to Dubai Police or UAE courts.

কোনো গ্রাহক যদি বকেয়া ঋণ ছেড়ে পালিয়ে যান এবং যোগাযোগ করা না যায়, তাহলে সিস্টেম স্বয়ংক্রিয়ভাবে একটি আনুষ্ঠানিক "ডিফল্ট নোটিশ" PDF তৈরি করবে — যা দুবাই পুলিশ বা আদালতে জমা দেওয়ার জন্য প্রস্তুত।

Firebase Configuration / ফায়ারবেস কনফিগারেশন

Create a .env file in the project root and add your Firebase credentials:

# Firebase Config — .env

VITE_FIREBASE_API_KEY=your_api_key

VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com

VITE_FIREBASE_PROJECT_ID=your_project_id

VITE_FIREBASE_STORAGE_BUCKET=your_bucket

VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id

VITE_FIREBASE_APP_ID=your_app_id

Security & Privacy

This system is private and owner-only. Firebase Authentication ensures that only authorised shop owners can access customer data. All data is stored in Firebase Firestore with strict security rules.

এই সিস্টেমটি সম্পূর্ণ ব্যক্তিগত ও মালিক-সুরক্ষিত। Firebase Authentication নিশ্চিত করে যে শুধুমাত্র অনুমোদিত দোকান মালিক গ্রাহক তথ্য অ্যাক্সেস করতে পারবেন।
